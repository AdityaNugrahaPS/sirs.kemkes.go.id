# Data SDM / Tenaga

[← Kembali ke README](../README.md)

Dataset jumlah tenaga kesehatan per jenis dan grup untuk setiap RS.

---

## Struktur Kolom

| Kolom | Deskripsi |
|-------|-----------|
| `kode_rs` | Kode RS |
| `grup_sdm` | Grup/kategori tenaga |
| `jenis_sdm` | Jenis tenaga (spesifik) |
| `jumlah_sdm` | Jumlah tenaga |
